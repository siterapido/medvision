# Tooling & Productivity Guide

## Recommended IDE
- **VS Code**: Strongly recommended due to TypeScript and Python support.

## Extensions
- **ESLint**: For realtime linting feedback.
- **Prettier**: For automatic formatting.
- **Python**: For backend development.
- **Tailwind CSS IntelliSense**: For class autocompletion.
- **AI Context**: For intelligent context management.

## Scripts & Automation

### Backend
- `scripts/run_backend.sh`: (If present) Helper to start the python service.
- **Makefiles**: Check root for any `Makefile` to simplify commands.

### Frontend
- `npm run dev`: Standard dev server.
- `npm run build`: Production build.
- `npm run lint`: Check for code issues.

## Debugging
- **React DevTools**: Essential for debugging component state.
- **Redux DevTools**: If using Redux (or similar state management).
- **Supabase Dashboard**: Use the local or cloud dashboard to inspect database tables and RLS policies.
